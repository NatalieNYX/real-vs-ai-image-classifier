# Real VS AI Generated Images Classifier 
CNN-based image classification model for detecting AI images

## Dataset

This project uses the **AI vs Real v4** dataset from **Roboflow Universe**.  

- **Dataset:** AI vs Real v4  
- **Source:** [Roboflow Universe](https://universe.roboflow.com/yolo-ai-art-v3/ai-vs-real-v_4)  
- **License:** Check dataset license on Roboflow (usually free for research/educational use)  
The dataset contains labeled images for **AI-generated vs real images** used to train the CNN classifier.
